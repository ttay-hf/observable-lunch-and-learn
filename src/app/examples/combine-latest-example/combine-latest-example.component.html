CombineLatest Example

This example shows off a few things:
  • How to use the combineLatest operator
  • Using "defer" with valueChanges so that the observable automatically emits when it's subscribed to
    • A form's valueChanges observable fires after its value has changed, but you can use concat/defer operators to emit a value immediately when the valueChanges observable is subscribed to

With combineLatest, the observable only fires the event after all of its combined observables have emitted at least once. You will notice with the default demo, the Combined Forms section only begins being updated after both fields have been modified. You can use the toggle button below the forms to setup the "Right field" to use concat/defer on that observable. When it is set to concat/defer, then the combined observable will start displaying as soon as the left field form is typed into.
